Pipeline ETL Completo con PySpark y MySQL


Este proyecto construye un pipeline completo de ETL para datos de películas de TMDB, transformando información cruda en un data warehouse MySQL con esquema en estrella. Incluye procesos de ingestión, transformación y carga de datos con PySpark, y expone un API para consultar la información procesada.

Se demuestran habilidades en ingeniería de datos, modelado de bases de datos analíticas y despliegue de pipelines reproducibles usando Docker y PySpark.

Estructura:
  • complete_ETL_PySpark
    • data
      • raw/ # Archivos JSON crudos
    • ingest
      • intake.py # Script de ingestión
    • spark_jobs
      • ETL scripts con PySpark
    • db
      • init SQL scripts # Inicialización de base de datos
    • utils
      • Funciones y módulos auxiliares
    • Dockerfile
    • docker-compose.yml
    • README.md
